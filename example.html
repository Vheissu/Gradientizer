Gradientizer

Simple CSS3 gradients support for jquery.

Gradientizer In Action

This is a horizontal gradient

            
              // The new cool way to write document.ready
              $(function() {

                  $("#horizontal").gradientizer({
                      direction: 'horizontal',
                      colors: [
                        {
                          '0%': 'red',
                          '10%': 'yellow',
                          '20%': 'pink',
                          '30%': 'green',
                          '40%': 'purple',
                          '50%': 'orange',
                          '70%': 'blue',
                          '80%': 'magenta',
                          '90%': 'cyan',
                          '100%': 'indigo'
                        }
                      ]
                  });

              });
